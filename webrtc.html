Instructions

  1. Start the signaling server: Run python3 webrtc-signaling-server.py in the project directory
  2. Open this page in two browser windows or tabs (same computer or different devices on same network)
  3. Enter the same Call ID/Room in both windows (e.g., "meeting123") - only peers with matching IDs will connect
  4. Optionally enter your name for identification
  5. In both windows, click "Connect to Server"
  6. In both windows, click "Start Local Camera" and allow camera access
  7. The peers in the same room will automatically connect via WebSocket signaling!
  8. You should see both local and remote video streams with WebGL cubic interpolation

Note: Only users who enter the same Call ID will be able to see and connect to each other. This allows multiple independent calls on the same server.
